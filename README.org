* Emacs Commands
** Remapping keys
*** Example
 
    + C-x C-u ==> upcase-region
    + remap keys to undo
    + (define-key global-map "\C-x\C-u" 'undo)

  | Keystrokes | Command Name                 |
  | - -------- | ------------                 |
  | C-x C-f    | find-file                    |
  | C-x C-v    | find-alternate-file          |
  | C-x i      | insert-file                  |
  | C-x C-s    | save-buffer                  |
  | C-x C-w    | write-file                   |
  | C-x C-c    | save-buffers-kill-emacs      |
  | C-h        | help-command                 |
  | C-h f      | describe-function            |
  | C-h k      | describe-key                 |
  | C-h t      | help-with-tutorial           |
